# Text Tools Project

A simple Python project for practicing file operations with `.txt` files.  
This repository contains utility functions for reading, writing, and manipulating text files.

## Features
- Write numbers from 1 to N into a file
- Read entire file content
- Read specific lines using `seek()` and `readline()`
- Count lines and characters in a file
- Merge multiple text files into one
- Search for words inside a file

## Project Structure

Working_with_files/

Run `demo.py` to see examples of how the functions work.
